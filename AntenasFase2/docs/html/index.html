Projeto Antenas - Fase 2: Grafos

Introdução

Nesta segunda fase, o projeto evolui para uma representação mais avançada: um grafo, onde cada antena é um vértice e as conexões são arestas entre antenas com a mesma frequência.

Esta estrutura permite aplicar algoritmos como DFS, BFS e encontrar caminhos entre antenas.

Objetivos

  • Representar antenas como vértices num grafo
  • Criar arestas entre antenas com a mesma frequência
  • Implementar procura em profundidade (DFS) e largura (BFS)
  • Listar todos os caminhos possíveis entre duas antenas
  • Detetar interseções entre frequências diferentes

Ficheiros

  • main.c: Execução dos testes e chamadas de função
  • funcoes.c: Lógica de grafos e algoritmos de procura
  • estruturas.h: Estruturas de grafo (Vértice, Aresta, Grafo)
  • mapa.txt: Exemplo de entrada com matriz de antenas
