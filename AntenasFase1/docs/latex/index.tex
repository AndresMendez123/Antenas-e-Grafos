\chapter{Projeto Antenas -\/ Fase 1\+: Listas Ligadas}
\hypertarget{index}{}\label{index}\index{Projeto Antenas -\/ Fase 1: Listas Ligadas@{Projeto Antenas -\/ Fase 1: Listas Ligadas}}
\hypertarget{index_intro_sec}{}\doxysection{\texorpdfstring{Introdução}{Introdução}}\label{index_intro_sec}
Esta fase do projeto consiste em representar uma cidade com várias antenas numa matriz, onde cada antena tem uma frequência específica. O objetivo é detetar localizações com {\bfseries{efeitos nefastos}} quando há alinhamento entre antenas da mesma frequência.

A representação das antenas e dos efeitos é feita com {\bfseries{listas ligadas}}.
