Projeto Antenas - Fase 1: Listas Ligadas

Introdução

Esta fase do projeto consiste em representar uma cidade com várias antenas numa matriz, onde cada antena tem uma frequência específica. O objetivo é detetar localizações com efeitos nefastos quando há alinhamento entre antenas da mesma frequência.

A representação das antenas e dos efeitos é feita com listas ligadas.

Objetivos

  • Representar antenas e localizações com estruturas ligadas
  • Inserir, remover e listar antenas de forma ordenada
  • Deduzir automaticamente os efeitos nefastos
  • Representar graficamente a matriz com antenas e efeitos
  • Guardar e carregar informação de ficheiros

Ficheiros

  • main.c: Ponto de entrada para testes e execução
  • funcoes.c: Implementações das funcionalidades com listas ligadas
  • estruturas.h: Estruturas de dados ANTENA e LOCAL
  • mapa.txt: Exemplo de matriz com antenas
  • Doxyfile: Configuração do Doxygen
